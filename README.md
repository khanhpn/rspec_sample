# Environment
- Ruby 3.0.0
- Rails 6.1.7
- Postgresql
# Setup
- Run `bundle install`
- Run `rails db:create`

# Setup database for development and test environment
- Run `rails db:migrate`
- Run `rails db:migrate RAILS_ENV=test`
- Run `rspec spec`

# Note
- Folder `spec/factories/` <- Put sample data
- Folder `spec/models/`  <- Write code rspec here
